配置Nginx Proxy Manager

通过http://<your_ip:81>打开浏览器查看服务

输入默认账号密码登录服务

登录后点击界面上的SSL Certificates给服务添加证书

随后依次点击Add SSL Certificates --> Let's Encrypt

输入自己的域名并同意条款后点击Save等待几分钟即可

随后点击Hosts --> Proxy Hosts

点击Add Proxy Host
